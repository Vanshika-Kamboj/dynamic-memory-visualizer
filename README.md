# 🧠 Dynamic Memory Management Visualizer

An interactive web-based tool to visualize core Operating System memory management techniques such as:

- 📦 Paging (FIFO Page Replacement)
- 🧩 Segmentation
- 💾 Virtual Memory

---

## 🚀 Features

- Visual simulation of Paging using FIFO algorithm
- Displays Page Hits and Page Faults
- Graphical representation using charts
- Segmentation visualization (Code, Stack, Heap, Data)
- Virtual Memory simulation
- Clean and modern UI

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

---

## 📊 How It Works

1. Enter memory size, page size, and number of frames
2. Run Paging simulation
3. Observe:
   - Page Hits & Faults
   - Frame changes
   - Graph visualization
4. Explore Segmentation and Virtual Memory modules

---

## 🎯 Learning Outcome

This project helps understand:

- Memory allocation techniques
- Page replacement algorithms
- Logical vs Physical memory
- OS memory optimization strategies
